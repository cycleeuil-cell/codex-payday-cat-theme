# 安全说明

## 隔离模型

安装器只读取本机 Microsoft Store `OpenAI.Codex` 包的程序文件，并把它们复制到用户自己的 `%LOCALAPPDATA%\CodexPaydayCat\runtime`。所有主题修改只发生在这个隔离副本内。Store 原版目录不会被写入。

主题版沿用 Codex 原来的用户资料目录，因此用户无需重新登录；脚本本身不会枚举、读取、复制、导出或上传账号资料、Cookie、聊天、任务、日志或数据库。

## 网络行为

主题 CSS 与状态宠物 JavaScript 不发起网络请求。安装/更新期间唯一预期的额外网络访问是 `npx` 或 `pnpm` 从 npm 获取 `@electron/asar` 打包工具。

## 公开仓库排除项

本仓库不得包含：

- Codex/ChatGPT 可执行文件或 Microsoft Store 包内容；
- 原始或修改后的 `app.asar`；
- 用户配置、登录凭据、Cookie、数据库、聊天、任务或日志；
- 来源电脑的安装状态、运行副本、备份和绝对用户路径。

`.gitignore` 会排除运行副本、备份、缓存、日志、安装状态和 ASAR 文件。发布前应再次执行敏感路径扫描与文件清单检查。

## 更新失败保护

安装器在找不到已知 WebView 入口、资源目录或脚本标记时会停止，不会继续强行修改。每个 Codex 版本的原始 `app.asar` 都单独备份，并记录 SHA-256。

## 报告问题

提交问题时不要附带账号信息、完整日志、聊天内容或 `app.asar`。只提供 Codex 版本、错误文字和已经脱敏的截图。
